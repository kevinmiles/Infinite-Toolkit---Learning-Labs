What you need to start using ITK sandbox

All the exercises related to Infinite Toolkit sandbox Learning Labs can be completed if

  • you are registered to DevNet
  • you fulfill the prerequisites described in this learning lab
  • you have a Cisco ID that works on Cisco.com and Cisco Communities.


Once you have access to CISCO domain, you can get more information about ITK there:

  • Cisco Communities
  • Developer Communities
